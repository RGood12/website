Workday | Time Off Fundamentals

Description

This four-day course includes lectures, demonstrations, and hands-on activities covering time off setup and administration. You will apply the core concepts of Workday Absence Management to:

  • Configure time off functionality
  • Set up time off calculations using the Workday calculation engine
  • Configure time off business processes
  • Perform time off administrative tasks

In addition, you will learn how to use standard and custom reports, as well as time off dashboards, and calendars to view time off related information.

This course will prepare you to set up and administer the Time Off function in Workday Absence Management, by covering:

  • Navigation and fuctioning withing Workday to perform time off-related tasks
  • Differntiating between time off and leave of absence
  • Configuring a time off plan and its components
  • Building calculations through the calculation engine to meet time off plan requirements
  • Using absence type groups and absence tables to manage time offs
  • Managing time off, including overrides, adjustments, and corrections
  • Editing time off business processes
  • Viewing time off-related information using standard and custom reports
  • Access absence-related dashboards and calendars to view time off information
